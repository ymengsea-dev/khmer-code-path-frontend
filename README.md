# AI-LMS Frontend

A modern **Next.js** (v16) web application that showcases a curated set of coding courses. The UI is built with **React 19**, **Tailwind CSS**, and a collection of reusable Shadcn UI components. It provides a clean, responsive interface for browsing courses, viewing detailed information, and launching an embedded IDE for hands‑on learning.

## Demo

![Course Detail Panel](https://github.com/your-repo/khmer-code-path-frontend/blob/main/.github/assets/demo.png?raw=true)

## Features

- **Course catalog** with rich metadata (institution, level, technologies, prerequisites).  
- Dynamic **detail panel** showing course description, achievement badge, and certificate preview.  
- **Embedded IDE** component for an in‑browser coding environment.  
- Reusable UI primitives from **shadcn/ui** (Button, Card, Badge, Sheet, etc.).  
- Dark‑mode support and smooth backdrop‑blur effects.

## Getting Started

### Prerequisites
- Node.js (≥ 20) or any compatible runtime (npm, Yarn, pnpm, bun).
- Git installed.

### Installation
```bash
# Clone the repository
git clone https://github.com/your-repo/khmer-code-path-frontend.git
cd khmer-code-path-frontend

# Install dependencies (pnpm example)
pnpm install
```

### Development
```bash
pnpm dev   # starts the Next.js dev server at http://localhost:3000
```
The app supports hot‑reloading; edit any file under `app/` or `components/` and the browser will refresh automatically.

### Building for Production
```bash
pnpm build   # generates an optimized production build
pnpm start   # runs the production server
```

### Linting
```bash
pnpm lint   # runs ESLint with the Next.js preset
```
